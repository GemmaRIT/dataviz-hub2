There are a lot of things to learn when you make a transition from Excel to R, especially if you haven't touched programming languages before, 
everything can be overwhelming and frustrating. Hopefully this article will help you make a smooth transition by helping you install and set up 
relevant software, explaining similarity and differences between Excel and R, and introducing some fundamental concepts of R.
